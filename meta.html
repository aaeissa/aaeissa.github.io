Changing Metadata with Python

Posted on Thu 06 July 2017 in Python

Easy method - python-docx module

from docx import Document

document = Document('test.docx')
document.core_properties.author
'real@email.com'

document.core_properties.created
datetime.datetime(2017, 5, 11, 19, 7)
 document.core_properties.last_modified_by
'goodguy@original.com'

document.core_properties.author = badguy@changed.com
document.core_properties.author
'badguy@changed.com'

document.save('meta.docx')

Formatted XML

<?xml version="1.0" encoding="UTF-8"?>
<cp:coreProperties xmlns:cp="http://schemas.openxmlformats.org/package/2006/metadata/core-properties" xmlns:dc="http://purl.org/dc/elements/1.1/" xmlns:dcmitype="http://purl.org/dc/dcmitype/" xmlns:dcterms="http://purl.org/dc/terms/" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
   <dc:title />
   <dc:subject />
   <dc:creator>badguy@changed.com</dc:creator>
   <cp:keywords />
   <dc:description />
   <cp:lastModifiedBy>ahmed@terbiumlabs.com</cp:lastModifiedBy>
   <cp:revision>1</cp:revision>
   <dcterms:created xsi:type="dcterms:W3CDTF">2017-05-11T19:07:00Z</dcterms:created>
   <dcterms:modified xsi:type="dcterms:W3CDTF">2017-05-11T19:07:00Z</dcterms:modified>
</cp:coreProperties>
